Questions and Answers

     
  1. About mirror.
    1. What is mirror?

      Mirror was designed to duplicate a directory hierarchy between two machines. It avoids copying files unnecessarily by comparing the file time-stamps and file sizes before transferring.

    2. Where do I get mirror?

      The latest version of mirror can always be found at:

      3.      ftp://sunsite.org.uk/packages/mirror/mirror.tar.gz  (or via http)
           ftp://sunsite.org.uk/packages/mirror/mirror.zip   (or via http)
       

    3. What's the latest version?

      As of 1 June 1998 the latest version of mirror is mirror-2.9.

    4. I am having a problem with mirror - what do I do?

      You should first try to run mirror in debug mode and see if this helps show up what the problem is.  Do this by calling mirror with "-d".   Extra "-d"'s will give increasing amounts of debugging messages.

      If you are using a version of perl below 5.004 then upgrade to a newer release.

      If you are still having a problem then email the author at lmjm@icparc.ic.ac.uk
